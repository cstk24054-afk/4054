# Smart ToDo（スマート・タスク管理アプリケーション）

ゲーム感覚のモチベーション機能（冒険者ステータス・XPシステム）と、ポモドーロタイマーを融合させた、直感的で高機能なタスク管理（ToDo）アプリケーションです。

## 📋 1. アプリケーション概要

### 🌟 主な機能
- **直感的なカンバンボード**: 「未着手」「進行中」「完了」の3レーン間をドラッグ＆ドロップで滑らかにタスク移動できます。
- **冒険者ステータスシステム**: タスクを完了するとXP（経験値）を獲得し、レベルアップや称号の変化を楽しめます。
- **ポモドーロタイマー連動**: 25分の集中セッションと5分の休憩を管理し、集中タスク完了時には追加XPを獲得できます。
- **インライン編集 & バリデーション**: タスク名のその場編集機能、および空文字での登録を防ぐ安全ガードを実装しています。
- **自動永続化**: すべてのデータはブラウザのLocalStorageに自動保存され、リロードしても状態が維持されます。

## 📋 2. 起動方法（ローカル開発環境）
手元のPC（ローカル環境）で本アプリケーションを起動・開発する手順です。

📦 前提条件
Node.js (v18以上推奨) がインストールされていること。

🏃 起動手順
依存パッケージのインストール

Bash
npm install
ローカル開発サーバーの起動

Bash
npm run dev
ブラウザでの確認
ターミナルに表示される http://localhost:5173/ にアクセスします。
